# Checkpoint — Responsive Website for Travel Agency & Fictional Restaurant

Two responsive single-page websites built with **HTML5**, **CSS3**, and **Bootstrap 5**:

| Page                       | File              | Description                                                                                                  |
| -------------------------- | ----------------- | ------------------------------------------------------------------------------------------------------------ |
| **Wanderlust Travels**     | `travel.html`     | A travel agency site showcasing popular destinations, curated tour packages (accordion), and a contact form. |
| **Ember & Basil — Foodie** | `restaurant.html` | A fictional restaurant site featuring a signature menu, about section, and reservation/contact form.         |

Both pages share a single stylesheet (`styles.css`) and are fully responsive across mobile, tablet, and desktop viewports.
